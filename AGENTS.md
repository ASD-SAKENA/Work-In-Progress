# راهنمای نویسندگی سند LaTeX

این راهنما برای کمک به دانشجویان در نوشتن صحیح و سازمان‌یافته‌ی سند LaTeX است.

## ساختار کلی سند

سند شامل بخش‌های زیر است:

1. **صفحه نام سند و اطلاعات درس** - اطلاعات درس، استاد، و دانشجویان
2. **صفحه عنوان** - عنوان پروژه و اطلاعات کلی
3. **فهرست مطالب** - فهرست خودکار
4. **فهرست جداول و شکل‌ها** - لیست خودکار
5. **فصل‌های اصلی** - محتوای پروژه
6. **منابع و واژه‌نامه** - منابع استفاده‌شده
7. **ضمائم** - مطالب اضافی
8. **تاریخچه تغییرات** - نسخه‌گذاری و تغییرات

## نحوه درج متن و شکل‌ها

### نوشتن فصل جدید

هر فصل جدید باید در دایرکتوری `chapters/` قرار داشته باشد.

**مثال:**
```latex
% فایل: chapters/methodology.tex
\chapter{\rl{روش‌شناسی}}

\section{\rl{مقدمه}}

در این بخش روش‌های استفاده‌شده را توضیح می‌دهیم...

\section{\rl{تجزیه و تحلیل}}

...
```

سپس در `thesis.tex` اضافه کنید:
```latex
\include{chapters/methodology}
```

### اضافه کردن شکل

```latex
\begin{figure}[h]
  \centering
  \includegraphics[width=0.8\textwidth]{figs/diagram.pdf}
  \caption{\rl{شرح شکل به فارسی}}
  \label{fig:diagram-label}
\end{figure}

\rl{همانطور که در شکل \ref{fig:diagram-label} دیده می‌شود...}
```

**نکات مهم:**
- عرض شکل را مناسب با متن تنظیم کنید (معمولاً 0.7-0.9 عرض صفحه)
- همیشه عنوان به فارسی درج کنید
- از `\label` برای ارجاع‌ دهی استفاده کنید

### اضافه کردن جدول

```latex
\begin{table}[h]
  \centering
  \caption{\rl{عنوان جدول}}
  \label{tab:table-label}
  \begin{tabular}{|c|c|c|}
    \hline
    \rl{ستون ۱} & \rl{ستون ۲} & \rl{ستون ۳} \\
    \hline
    مقدار & مقدار & مقدار \\
    \hline
  \end{tabular}
\end{table}

\rl{جدول \ref{tab:table-label} نتایج را نشان می‌دهد...}
```

**نکات مهم:**
- از `\rl{}` برای متن فارسی استفاده کنید
- هدرهای جدول را با `\bf` پررنگ کنید
- تعداد ستون‌ها باید با تعریف `tabular` مطابق باشد

## ترجمه و عناوین

### شماره‌گذاری فارسی

برای استفاده از شماره‌های فارسی (۱، ۲، ۳ به جای 1, 2, 3)، از کلید `\rl{}` استفاده کنید:

```latex
\section{\rl{بخش ۱: مقدمه}}

در فصل \ref{ch:intro} مقدمه را بررسی کردیم...
```

### عناوین فارسی

هر فصل باید شروع شود با:
```latex
\chapter{\rl{نام فصل به فارسی}}

\section{\rl{نام بخش به فارسی}}

\subsection{\rl{نام زیربخش به فارسی}}
```

### مراجع و منابع

منابع در فایل `bibs/bibs.tex` مدیریت می‌شوند:

```latex
\include{bibs/bibs}
```

منابع را در فرمت BibTeX درج کنید:

```bibtex
@article{agile2020,
  author = {نویسنده},
  title = {عنوان مقاله},
  journal = {نام مجله},
  year = {2020},
  pages = {۱۰-۲۰}
}

@book{scrum2018,
  author = {نویسنده},
  title = {عنوان کتاب},
  publisher = {ناشر},
  year = {2018}
}
```

سپس در متن از آن استفاده کنید:

```latex
\rl{در تحقیقات قبلی} \cite{agile2020} \rl{نشان داده شده است...}
```

## به‌روزرسانی متادیتا

### تغییر نام سند

فایل `front/info.tex` را ویرایش کنید:

```latex
\newcommand{\DocumentNameFa}{سند نام جدید}
\newcommand{\DocumentNameEn}{New Document Name}
\newcommand{\ThesisTitle}{عنوان پروژه جدید}
```

### تغییر ماه و سال

```latex
\newcommand{\ThesisMonth}{مهر}
\newcommand{\ThesisYear}{۱۴۰۶}
```

### اضافه کردن استاد مشاور

```latex
\newcommand{\ThesisAdvisor}{دکتر نام استاد}
```

## تاریخچه تغییرات

در هر تغییر مهم، فایل `front/changelog.tex` را به‌روزرسانی کنید:

```latex
\begin{table}[h]
  \centering
  \caption{\rl{تاریخچه نسخه‌های سند}}
  \begin{tabular}{|c|c|c|p{5cm}|}
    \hline
    \rl{\bf نسخه} & \rl{\bf تاریخ} & \rl{\bf نویسندگان} & \rl{\bf توضیحات تغییرات} \\
    \hline
    ۱.۰ & \rl{خرداد ۱۴۰۵} & \rl{تمام دانشجویان} & \rl{نسخه اولیه} \\
    ۱.۱ & \rl{تیر ۱۴۰۵} & \rl{معین آعلی} & \rl{افزودن فصل روش‌شناسی} \\
    \hline
  \end{tabular}
\end{table}
```

## کامپایل سند

برای کامپایل سند LaTeX استفاده کنید:

```bash
# کامپایل یکبار
xelatex thesis.tex

# کامپایل دوبار برای TOC و references
xelatex thesis.tex
xelatex thesis.tex
```

اگر از منابع BibTeX استفاده می‌کنید:

```bash
xelatex thesis.tex
bibtex thesis
xelatex thesis.tex
xelatex thesis.tex
```

## بهترین‌های

### سبک و نگارش

1. **صحیح‌سازی هجایی:** قبل از تسلیم، متن را برای اشتباهات هجایی بررسی کنید.
2. **سازگاری استایل:** از یک سبک شماره‌گذاری و نام‌گذاری برای تمام فصل‌ها استفاده کنید.
3. **فاصل‌گذاری:** بین پاراگراف‌ها و بخش‌ها فاصله مناسب گذاشتن اهمیت دارد.

### مراجع

4. **مراجع کامل:** نام کامل نویسندگان، سال انتشار، و نام ناشر را درج کنید.
5. **سبک یکپارچه:** برای تمام منابع از یک سبک استفاده کنید (به عنوان مثال APA یا IEEE).
6. **ارجاع دادن:** برای تمام گفتارهای دیگری و نتایج تحقیقات دیگران، مراجع درج کنید.

### جداول و شکل‌ها

7. **عنوان واضح:** هر جدول و شکل باید عنوان واضح و شرح مختصر داشته باشد.
8. **ارجاع‌دهی:** در متن، برای تمام جداول و شکل‌ها ارجاع دهید.
9. **کیفیت:** از شکل‌های عالی‌الجودت (حداقل 300 DPI) استفاده کنید.

## نکات فنی

- **Document Class:** سند از `book` document class استفاده می‌کند.
- **زبان:** برای متن فارسی از بسته `xepersian` استفاده می‌شود.
- **فونت:** فونت پیش‌فرض برای متن فارسی تنظیم شده است.
- **سطح‌های عنوان:** `\chapter`, `\section`, `\subsection` استفاده کنید.

## مثال کامل فصل

```latex
% فایل: chapters/introduction.tex

\chapter{\rl{مقدمه}}

\section{\rl{زمینه‌ی مسئله}}

\rl{مقدمه‌ای بر مسئله...}

\section{\rl{اهداف پروژه}}

\rl{اهداف اصلی پروژه:}

\begin{enumerate}
  \item \rl{هدف اول}
  \item \rl{هدف دوم}
  \item \rl{هدف سوم}
\end{enumerate}

\section{\rl{ساختار سند}}

\rl{این سند بدین‌صورت سازمان‌یافته است:}

\begin{itemize}
  \item \rl{فصل \ref{ch:methodology}: روش‌شناسی}
  \item \rl{فصل \ref{ch:results}: نتایج}
  \item \rl{فصل \ref{ch:conclusions}: نتیجه‌گیری}
\end{itemize}
```
